/gameinfo
    - вместе с командой должен идти id игры или название, которое будет смотреть через python-steam-api 
    - выводить название, дату выхода, картинку, краткое описание, мб долю положительных оценок
    - если id, то ищем сначала в базе, потом через api
    - если название - сразу через api 
/similar
    - нужен id игры (надо будет проверить на тип - app, dlc или что-то ещё)
/trends
    - скорее всего имеет смысл брать релизы последней недели со steam db с фильтрами только на игры (без бет, дополнений и подобного)
    - можно брать первые 5-10 элементов (опционально)
/new_friends_games
    - просто выводит информацию о новых добавленных играх с пометками о свежести игр (этот год, этот месяц)
